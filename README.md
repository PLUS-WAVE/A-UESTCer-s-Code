### 综设（进阶式挑战性综合项目）

从大二上开始持续三学期到大三上结束的综合设计课程😅：

> 开发一个基于多视觉融合的三维重建平台，培养学生对多视角观察和三维重建领域的深入理解。通过硬件平台的需求分析以及相关选型，提供学生实践机会，通过完成任务，掌握三维重建的关键技术和方法。
>
> 三个阶段最终实现一个基于嵌入式平台的多视觉三维立体重建系统
