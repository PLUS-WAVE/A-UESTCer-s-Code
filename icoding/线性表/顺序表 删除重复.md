## 题目：

编写算法，在一非递减的顺序表L中，删除所有值相等的多余元素。要求时间复杂度为O(n)，空间复杂度为O(1)。

函数原型如下：  
`void del_dupnum(SeqList *L)`

相关定义如下：

```c
struct _seqlist{
    ElemType elem[MAXSIZE];
    int last;
};
typedef struct _seqlist SeqList;
```

## 代码：

```cpp
#include <stdio.h>
#include <stdlib.h>
#include "list.h" // 请不要删除，否则检查不通过

void del_dupnum(SeqList *L)  {
    int j = 0;
    int i = 0;
    while (j <= L->last) {
        if (L->elem[j] > L->elem[i]) {
            L->elem[i + 1] = L->elem[j];
            i++;
        }
        else {
            j++;
        }
    }
    L->last = i ;
}
```
